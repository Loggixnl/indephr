# IndepHR

## Project Overview
HR website built with Vue 3, Tailwind CSS, deployed on Vercel.

## Tech Stack
- **Framework:** Vue 3 (Composition API)
- **Styling:** Tailwind CSS
- **Build Tool:** Vite
- **Deployment:** Vercel

## Project Structure
```
/src
  /assets        # Static assets (images, fonts)
  /components    # Reusable Vue components
  /composables   # Vue composables (shared logic)
  /layouts       # Page layouts
  /pages         # Page components
  /router        # Vue Router configuration
  /stores        # Pinia stores (state management)
  /styles        # Global styles
  App.vue        # Root component
  main.ts        # Application entry point
```

## Development Commands
```bash
npm install      # Install dependencies
npm run dev      # Start dev server
npm run build    # Production build
npm run preview  # Preview production build
```

1. Introduction
This style guide outlines the visual and communicative principles for the IndepHR brand. The core mission is to close the feedback loop with transparent communication and concrete follow-up, thereby building trust, driving engagement, and transforming feedback into positive change for both employers and employees.

4. Typography
Typography plays a crucial role in conveying IndepHR's professional yet approachable tone. We will utilize a clean, modern sans-serif typeface for all communications, ensuring readability and a contemporary feel.
Primary Typeface: Montserrat
Montserrat is a geometric sans-serif typeface inspired by old posters and signs from the traditional Montserrat neighborhood of Buenos Aires. It is highly versatile, legible, and has a friendly yet professional appearance, making it ideal for both headlines and body text. Its wide range of weights allows for clear hierarchy and visual interest.

Usage: Headlines, subheadings, and primary body text.
Key Characteristics: Geometric, clean, modern, highly legible, versatile.
Secondary Typeface: Open Sans
Open Sans is a humanist sans-serif typeface designed for a neutral yet friendly appearance. It is optimized for print, web, and mobile interfaces, and has excellent legibility characteristics. It pairs well with Montserrat, providing a slightly softer and more open feel for longer passages of text or supplementary information.

Usage: Body text, captions, UI elements, and longer content blocks.
Key Characteristics: Humanist, open, friendly, highly legible, web-optimized.
5. Imagery and Graphics
Imagery for IndepHR should reflect our human-centric, intelligent, and forward-thinking approach. We aim for visuals that are authentic, diverse, and convey a sense of collaboration, positive impact, and data-driven insights.
Photography Style
Photography should be authentic and relatable, focusing on real people in professional yet approachable settings. Images should convey collaboration, engagement, and positive interactions within a workplace environment. Natural lighting and clean, uncluttered backgrounds are preferred to maintain a modern and professional aesthetic.

Themes: Diverse teams collaborating, individuals engaged in thoughtful work, HR professionals interacting with employees, moments of genuine connection and feedback.
Composition: Clean, well-lit, and natural. Avoid overly staged or corporate stock photos. Focus on candid or semi-candid shots that evoke genuine emotion and connection.
Color & Tone: Bright, optimistic, and consistent with the brand's color palette. Avoid overly saturated or desaturated images.
Illustration Style
Illustrations should be minimalist, abstract, and utilize the IndepHR color palette. They can be used to simplify complex concepts, visualize data, or add a friendly touch to interfaces and marketing materials. Line art with subtle gradients or flat shapes are preferred, maintaining a modern and sophisticated feel.

Themes: Abstract representations of data flow, feedback loops, growth, connection, and AI integration. Human figures in illustrations should be stylized and inclusive.
Style: Clean lines, geometric shapes, and a limited color palette derived from the brand colors. Avoid overly detailed or cartoonish illustrations.
Purpose: To explain concepts, enhance user interfaces, and add visual interest without distracting from the core message.
Iconography
Icons should be simple, clear, and consistent in style. A flat or subtly outlined style is recommended, using the primary brand colors. Icons should be easily recognizable and support intuitive navigation and understanding within the platform and marketing materials.
6. Tone of Voice
IndepHR's tone of voice is professional, empathetic, insightful, and empowering. We communicate with clarity and confidence, always aiming to educate, support, and inspire our audience. Our language is accessible, avoiding overly technical jargon where possible, while maintaining authority in the HR tech space.

Professional: Maintain a high standard of professionalism in all communications. Use clear, concise language.
Empathetic: Show understanding and consideration for the challenges faced by HR professionals and employees. Frame solutions in a supportive and helpful manner.
Insightful: Provide valuable, data-driven perspectives. Demonstrate expertise in HR and feedback management.
Empowering: Encourage action and positive change. Highlight how IndepHR enables users to achieve their goals and make a difference.
Accessible: Use straightforward language. Explain complex concepts simply without oversimplifying.

Examples of Language:

Instead of: "Leverage our proprietary algorithms for optimal feedback aggregation."

Use: "Our intelligent AI analyzes feedback to give you clear, actionable insights."

Instead of: "Implement our cutting-edge employee engagement solution."

Use: "Build a more engaged workforce with our intuitive feedback platform."
7. UI/UX Principles
IndepHR's user interface and experience are designed to be intuitive, efficient, and delightful. Our principles ensure that users can easily navigate the platform, understand insights, and take action effectively.

Clarity: Information is presented clearly and concisely, minimizing cognitive load. Visual hierarchy guides users through content.
Consistency: UI elements, interactions, and terminology are consistent across the platform, fostering familiarity and ease of use.
Efficiency: Workflows are streamlined, allowing users to complete tasks quickly and effectively. Automation is leveraged to reduce manual effort.
Accessibility: The platform is designed to be usable by individuals with diverse abilities, adhering to accessibility standards (e.g., WCAG).
Feedback & Guidance: Users receive clear feedback on their actions and are guided through complex processes with helpful prompts and explanations.
Visual Appeal: A clean, modern aesthetic that aligns with the brand's visual identity, using the defined color palette and typography to create an engaging experience.
